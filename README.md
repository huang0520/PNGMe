# PNGMe

A Rust project for learning systems programming by implementing a PNG parser/encoder from scratch.

**Based on [PNGme: An Intermediate Rust Project](https://jrdngr.github.io/pngme_book/) by jrdngr**

## Quick Start

Build the project:
```bash
cargo build
```

Run test:
```bash
cargo test
```

Run the project:
```bash
cargo run
```
